Ciphertags
==========
Implementation of the board game "Codenames" as a webapp, using
 - React
 - Redux
 - Socket.io
 - ES6

Roughly following the excellent tutorial at http://teropa.info/blog/2015/09/10/full-stack-redux-tutorial.html

Getting Started
---------------
Using nodejs v4.0.0 or above:

`npm install`

`npm run test`
